# Eduvestia Elite

Application d'intelligence financière premium (Freemium/Premium).

## Fonctionnalités
- **Studio IA** : Génération d'ebooks personnalisés avec audit fiscal.
- **Social** : Chat global, salons par pays, messagerie privée 1v1.
- **Lounge** : Quiz compétitif (type Kahoot) multi-joueurs.
- **Simulateurs** : DCA, Impôts 2026, Matelas de sécurité, etc.
- **Sync Cloud** : Intégration Supabase pour l'authentification et la sauvegarde.

## Déploiement sur Vercel
1. Connectez votre dépôt GitHub à Vercel.
2. Ajoutez les variables d'environnement suivantes :
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
3. Déployez !
